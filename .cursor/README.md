# .cursor — Cursor AI config for HexaStack Solutions

- **`rules/`** — Cursor rules (00-MASTER, 01–21 domain rules, 21-AGENT-CHAIN for the pipeline).
- **`skills/`** — One folder per pipeline agent; each has `SKILL.md` (role, inputs, output, principles).

**To run the agent pipeline:** Read `AGENTS.md` and `rules/21-AGENT-CHAIN.md`. Use `skills/<agent-name>/SKILL.md` for per-agent behavior.

**Project layout:** See repo root `PROJECT_STRUCTURE.md`.
