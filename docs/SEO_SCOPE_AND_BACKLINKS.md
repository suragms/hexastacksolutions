## 4. Backlinks – high‑ranking strategy

- **Track** all backlinks in Admin (URL, source site, DA/DR if known, date, notes).
- **Prioritise:** .edu, .gov, quality business directories, Gulf/Kerala local directories, tech and startup listings.
- **Content:** Guest posts, directory submissions, partner links, case studies, press.
- **Avoid:** Spam link farms, irrelevant sites.
- **Admin:** Use the **Backlinks** tab to add, edit, and track each link (URL, source, type, status) for full control.

## 5. Admin control

- **Backlinks:** Admin can add, edit, delete backlinks and track status/notes.
- **Dashboard:** Analytics, enquiries, portfolio, settings, services, products, **SEO Pages**, **Backlinks** – all in one place for full control.
